# Portfólio Pessoal - Atividade Prática UNINTER

Este projeto consiste no desenvolvimento de um portfólio online pessoal, realizado como atividade prática para a disciplina de **Aplicações Web e Móveis**. O objetivo é aplicar conceitos fundamentais de desenvolvimento front-end sem o uso de frameworks.

## 📋 Requisitos do Projeto
O desenvolvimento seguiu as diretrizes obrigatórias da instituição:

* **Tecnologias puras:** Uso exclusivo de HTML5, CSS3 e JavaScript (Vanilla).
* **Sem Frameworks:** Proibido o uso de Bootstrap, Tailwind, React, jQuery ou similares.
* **Estrutura:** Site funcional com 4 seções principais interligadas por um menu fixo.
* **Hospedagem:** Código versionado no GitHub e publicado via GitHub Pages.

---

## 💻 Seções do Portfólio
1.  **Sobre mim:** Apresentação pessoal, hobbies e trajetória profissional.
2.  **Formação:** Detalhes sobre a graduação em Publicidade e Propaganda e o atual curso de Engenharia de Software.
3.  **Portfólio:** Exposição de projetos realizados (incluindo trabalhos da Photolith).
4.  **Contato:** Formulário com validação de campos e simulação de envio via JavaScript.

---

## 🛠️ Implementações Técnicas (JavaScript)
Para atender aos critérios de avaliação, foram implementadas as seguintes funcionalidades em JavaScript puro:

* **Validação de Formulário:** Verificação de campos vazios e validação de formato de e-mail. 
* **Simulação de Envio:** Limpeza de campos e exibição de alerta de sucesso após o "envio".
* **Interatividade:** Foi criado o botão para tema escuro e claro.
